# 📚 E-Learning

A full-stack **MERN** (MongoDB, Express.js, React.js, Node.js) based **E-Learning Platform** designed to streamline online education. It enables instructors to create and manage courses while allowing students to enroll, learn, and track their progress seamlessly.

---

🚀 Features

- 👩‍🏫 Instructor dashboard to create, edit, and manage courses
- 🎓 Student dashboard with course enrollment and progress tracking
- 📺 Course content: video lectures
- 🔐 Secure user authentication and authorization (JWT-based)
- 📊 Admin panel for user and course management


## 🛠️ Tech Stack

### Frontend:
- React.js
- Tailwind CSS / ShadCn (UI styling)

### Backend:
- Node.js
- Express.js
- MongoDB with Mongoose

### Other:
- JSON Web Tokens (JWT) for authentication
- Multer for file uploads (e.g. videos)
- Cloudinary for media storage (profile)
